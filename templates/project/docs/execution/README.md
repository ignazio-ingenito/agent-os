# Execution state

status: none

Questo file è il puntatore operativo al lavoro attivo.

Usare `status: none` quando non esiste una wave attiva. Quando esiste una wave attiva, indicare qui il percorso o riferimento della wave. Non dedurre il lavoro attivo da nome, data o ultima modifica.

Alla chiusura della wave, aggiornare questo file nella stessa modifica che registra la chiusura.
